Fake-News Sentiment & Topic Modelling¶

NLP analysis of fake-news text: transformer-based sentiment scoring (DistilBERT) and Latent Dirichlet Allocation topic modelling, with interpretation of the resulting topics.

Key Insights¶

Task 1A – Sentiment Analysis

Using the HuggingFace Transformers pipeline with DistilBERT (distilbert-base-uncased-finetuned-sst-2-english), I computed sentiment labels (POSITIVE/NEGATIVE) and confidence scores for each article in the fakenews dataset. The key finding is that spam/fake messages exhibit a higher proportion of POSITIVE sentiment than normal/real messages. This reflects the tendency of fake news to use emotionally charged, sensational, and provocative language designed to trigger strong emotional reactions — a pattern a rule-based tool like TextBlob would miss because it cannot understand word order or negation.

Task 1B – Topic Modelling

Using Latent Dirichlet Allocation (LDA) on a CountVectorizer document-term matrix, I identified the optimal number of latent topics by evaluating perplexity across k = 2 to 10. The key finding is that certain topics are predominantly associated with fake/spam content (characterised by celebrity, political outrage, and sensational vocabulary), while others are predominantly real/normal (characterised by factual, event-driven reporting language). This demonstrates that topic structure alone carries meaningful signal for distinguishing fake from real news.

Personal Takeaway

This project revealed how modern transformer-based models (DistilBERT) dramatically outperform older rule-based approaches (TextBlob) by understanding context, negation, and sentence structure — not just individual word scores. It also showed that unsupervised topic modelling can surface meaningful content clusters from raw text without any labelled training data, which has direct applications in misinformation detection at scale.

from scipy.stats import chi2_contingency
import numpy as np

df['sentiment_binary'] = (df['sentiment_label'] == 'POSITIVE').astype(int)

sentiment_summary = df.groupby('label').agg(
    pct_positive=('sentiment_binary', 'mean'),
    mean_score=('sentiment_score', 'mean'),
    count=('label', 'count')
).round(4)
sentiment_summary.index = sentiment_summary.index.map({0: 'Real/Normal', 1: 'Fake/Spam'})
print(sentiment_summary)

contingency = pd.crosstab(df['label'], df['sentiment_label'])
chi2, p, dof, expected = chi2_contingency(contingency)
print(f'\nChi-square = {chi2:.2f}, p-value = {p:.4e}')
print(f"Result: {'Statistically significant' if p < 0.05 else 'Not significant'} (p < 0.05)")

def preprocess(text):
    text = str(text).lower()
    text = re.sub(r'http\S+|www\S+', ' ', text)
    text = re.sub(r'[^a-z\s]', ' ', text)
    return ' '.join(w for w in text.split() if w not in STOPWORDS and len(w) > 2)

df['clean_text'] = df['text'].apply(preprocess)
print('Pre-processing done.')

from sklearn.decomposition import LatentDirichletAllocation

k_range = range(2, 11)
perplexities   = []
log_likelihoods = []

for k in k_range:
    lda_tmp = LatentDirichletAllocation(
        n_components=k, random_state=42, max_iter=10,
        learning_method='online', batch_size=256
    )
    lda_tmp.fit(dtm)
    perplexities.append(lda_tmp.perplexity(dtm))
    log_likelihoods.append(lda_tmp.score(dtm))
    print(f'k={k:2d}  perplexity={perplexities[-1]:.1f}  log-likelihood={log_likelihoods[-1]:.0f}')

best_k = list(k_range)[int(np.argmin(perplexities))]
print(f'\n>>> Optimal number of topics (lowest perplexity): k = {best_k}')
