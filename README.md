## How to read this repo

- **Fastest:** open a project folder and click the `*.html` report — it renders in the browser via GitHub Pages, or downloads as a viewable file otherwise.
- **Source:** the matching `*.ipynb` is the working notebook.
- **Context:** each project's `_README.md` states the objective, methods, and what to look at.

If a `.ipynb` preview is blank on GitHub (a known GitHub issue, not a problem with the notebook), use the paired `.html` file.
